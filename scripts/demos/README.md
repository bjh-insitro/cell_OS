# Demo Scripts

Example scripts demonstrating Cell OS capabilities.

## Active Demos

- `create_custom_design.py` - Create custom experimental designs
- `demo_bio_vm_improvements.py` - Demo biological VM improvements

## Usage

Run demos directly:

```bash
python scripts/demos/[demo_name].py
```

## Archived Demos

Old demo scripts have been moved to `scripts/archive/demos/` including:
- Old imaging loop demos (superseded)
- Old POSH campaign demos (superseded)
- Legacy loop demos (run_loop.py, run_loop_v2.py)
