# Debugging Scripts

Diagnostic and debugging utilities for troubleshooting Cell OS components.

## Scripts

- `check_db_consumables.py` - Check database consumables inventory
- `debug_recipe.py` - Debug recipe generation issues
- `debug_workflow.py` - Debug workflow execution
- `diagnose_posh_optimizer.py` - Diagnose POSH optimizer issues
- `diagnose_score_landscape.py` - Visualize acquisition score landscape
- `generate_synthetic_embeddings.py` - Generate synthetic embeddings for testing

## Usage

These scripts are for development and debugging only. Run them directly:

```bash
python scripts/debugging/[script_name].py
```
